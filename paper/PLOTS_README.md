# Benchmark Plots

This directory contains plots generated by the benchmark calculation script.

## Generated Plots

After running `run_benchmarks.clj`, the following plots will be generated:

1. **`16Opd_dcs.png`** - 16O(p,d)15O transfer reaction differential cross section
2. **`11Li_dd_elastic.png`** - 11Li(d,d) elastic scattering
3. **`11Li_dd_inelastic.png`** - 11Li(d,d') inelastic scattering (monopole L=0)
4. **`C12_alpha_alpha_prime.png`** - ¹²C(α,α')¹²C* inelastic scattering (2⁺ state)
5. **`summary.png`** - Summary plot showing all reactions together

## Plot Format

- **X-axis**: Scattering angle θ (degrees)
- **Y-axis**: Differential cross section dσ/dΩ (mb/sr)
- **Format**: PNG, 1000×600 pixels

## Viewing Plots

Plots can be viewed with any image viewer or in a web browser. They are saved in PNG format for easy inclusion in papers and presentations.

## Regenerating Plots

To regenerate all plots, simply run:

```bash
./paper/run_benchmarks.sh
```

or

```bash
clj -M -e "(load-file \"paper/run_benchmarks.clj\")" -e "(paper.run-benchmarks/run-all-benchmarks)"
```
