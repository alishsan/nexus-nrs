# Changelog

All notable changes to the Nexus-NRS project.

## [0.1.0] - Unreleased

### Added

- Initial Nexus-NRS (Nuclear Reaction Suite) release.
- Scattering: phase shifts, R-matrix, optical potentials.
- Transfer reactions: DWBA (POST), bound-state overlap.
- Inelastic scattering and halo nuclei support.
- Web dashboard for interactive calculations.
- Paper data generators: `generate_paper_data.clj`, `generate_halo_numerov_comparison.clj`.
