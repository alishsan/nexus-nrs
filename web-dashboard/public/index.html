Elastic Scattering

Computes single-channel elastic scattering for a chosen projectile + target, using the Woods–Saxon parameters above as the effective optical potential for that system. Default preset matches α + ¹⁴⁸Sm at Elab = 50 MeV (real [65, 7.5, 0.67] MeV/fm + imaginary volume WS W₀ = 30 MeV via *elastic-imag-ws-params*). Elastic dσ sums nuclear partial waves L = 0 … Lmax (default 41). Large Lmax with a strong real well plus Coulomb can inflate fN numerically (e.g. p+¹⁶O dσ/σRuth ≫ 10³); use V0=R0=0 to verify ratio ≡ 1. The main Angular momenta field is for phase shift / R-matrix / inelastic only.

Inelastic Scattering Parameters

Projectile + target define the reaction (e.g. p + ¹²C). Target sets typical E_ex and β (λ=2).
